# 部署文档

## 一、环境要求
- JDK 17+（开发使用 JDK 25）
- Maven 3.6+
- SQLite3（无需单独安装服务，驱动内置）
- Linux 服务器（生产）/ Windows（开发）

## 二、本地开发运行
```bash
cd autism-screening-backend
mvn spring-boot:run
# 服务启动于 http://localhost:8081
```
首次启动自动：
1. 创建 `data/` 与 `data/videos/` 目录
2. 建立 SQLite 数据库 `data/autism.db`
3. 执行 `schema.sql` 建表 + `data.sql` 写入种子数据
4. 创建默认管理员 `admin / admin123`

## 三、打包部署（Linux 服务器）

### 3.1 构建 jar
```bash
mvn clean package -DskipTests
# 产物：target/autism-screening-backend.jar
```

### 3.2 上传至服务器
目录结构建议：
```
/opt/autism/
├── autism-screening-backend.jar
├── data/
│   ├── autism.db          # 自动生成
│   └── videos/            # 上传 q1.mp4 ~ q20.mp4
└── application.yml        # 可选：外部配置覆盖
```

### 3.3 后台启动
```bash
nohup java -jar /opt/autism/autism-screening-backend.jar \
  --spring.datasource.url=jdbc:sqlite:/opt/autism/data/autism.db \
  --autism.video-dir=/opt/autism/data/videos \
  > /opt/autism/app.log 2>&1 &
```

### 3.4 Systemd 服务（推荐）
`/etc/systemd/system/autism.service`：
```ini
[Unit]
Description=Autism Screening Backend
After=network.target

[Service]
Type=simple
User=autism
WorkingDirectory=/opt/autism
ExecStart=/usr/bin/java -jar /opt/autism/autism-screening-backend.jar
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
```
```bash
systemctl daemon-reload
systemctl enable --now autism
systemctl status autism
```

## 四、Nginx 反向代理 + HTTPS（微信小程序要求 HTTPS）
```nginx
server {
    listen 443 ssl;
    server_name api.yourdomain.com;

    ssl_certificate     /etc/nginx/ssl/server.pem;
    ssl_certificate_key /etc/nginx/ssl/server.key;

    client_max_body_size 50M;

    location /api/video/ {
        # 视频防盗链（可选）
        valid_referers none blocked yourdomain.com;
        proxy_pass http://127.0.0.1:8081;
        proxy_set_header Range $http_range;
    }

    location /api/ {
        proxy_pass http://127.0.0.1:8081;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 五、微信小程序配置

### 5.1 服务器域名白名单
登录 [微信公众平台](https://mp.weixin.qq.com) → 开发管理 → 开发设置 → 服务器域名：
- **request 合法域名**：`https://api.yourdomain.com`
- **download 合法域名**（视频）：`https://api.yourdomain.com`

### 5.2 小程序调用示例
```javascript
// 封装请求
const BASE = 'https://api.yourdomain.com'
function request(url, method, data) {
  return wx.request({
    url: BASE + url,
    method,
    data,
    header: { 'X-Token': wx.getStorageSync('token') || '' }
  })
}

// 微信登录
wx.login({
  success: res => {
    request('/api/user/wx-login', 'POST', { code: res.code })
      .then(r => wx.setStorageSync('token', r.data.token))
  }
})
```

### 5.3 生产环境对接微信登录
`UserService.wxLogin` 中将演示逻辑替换为真实调用：
```java
// GET https://api.weixin.qq.com/sns/jscode2session
//   ?appid=APPID&secret=SECRET&js_code=CODE&grant_type=authorization_code
// 返回 { openid, session_key }
```
将 `appid`/`secret` 配置在 `application.yml`。

## 六、配置项说明（application.yml）
| 配置 | 默认值 | 说明 |
|------|--------|------|
| `server.port` | 8081 | 服务端口 |
| `spring.datasource.url` | `jdbc:sqlite:data/autism.db` | 数据库路径 |
| `autism.video-dir` | `data/videos` | 视频目录 |
| `autism.demo-code` | `123456` | 演示验证码 |
| `autism.token-expire-hours` | 72 | Token 有效期 |
| `autism.allowed-referer` | 空 | 视频防盗链 Referer（空=不校验） |

## 八、数据库备份
```bash
# 冷备份（停止服务后）
cp /opt/autism/data/autism.db /backup/autism-$(date +%F).db
# 或使用 sqlite3 在线备份
sqlite3 /opt/autism/data/autism.db ".backup /backup/autism-$(date +%F).db"
```
建议加入 crontab 定时备份。
