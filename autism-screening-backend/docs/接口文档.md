# 接口文档

**Base URL**：`http://<服务器地址>:8081`
**统一响应格式**：
```json
{ "code": 0, "message": "success", "data": ... }
```
`code=0` 成功；`code!=0` 失败（`code=401` 未登录、`code=403` 无权限、`code=400` 参数错误）。

**鉴权方式**：登录返回 `token`，后续请求在 Header 携带 `X-Token: <token>`（也可用 query 参数 `?token=`）。视频接口无需 token。

> 微信小程序调用示例：`wx.request({ url, header:{ 'X-Token': token }, ... })`

## 一、用户接口 `/api/user`

### 1.1 发送验证码 `POST /api/user/send-code`
公开。请求体：`{ "phone": "13800000001" }`
响应：`{ "sent": true, "demoCode": "123456", "tip": "演示模式验证码：123456" }`
> 演示模式验证码固定为 `123456`，配置项 `autism.demo-code`。

### 1.2 注册 `POST /api/user/register`
公开。请求体：
```json
{ "phone": "13800000001", "code": "123456", "password": "abc123", "passwordConfirm": "abc123" }
```
响应：`LoginResponse`（含 token）

### 1.3 验证码登录 `POST /api/user/login`
公开。`{ "phone": "13800000001", "code": "123456" }` → `LoginResponse`

### 1.4 微信小程序登录 `POST /api/user/wx-login`
公开。`{ "code": "wx.login()返回的code" }` → `LoginResponse`
> 演示模式以 code 生成虚拟 openid；生产环境需在 `UserService.wxLogin` 中对接微信 `jscode2session` 接口。

### 1.5 退出登录 `POST /api/user/logout`
需登录。无请求体。

### 1.6 个人信息 `GET /api/user/profile`
需登录。返回当前用户信息。

## 二、儿童档案接口 `/api/child`（均需登录）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/child/list` | 当前用户儿童列表 |
| POST | `/api/child/add` | 新增（年龄限制 1-5 岁） |
| PUT | `/api/child/update/{id}` | 修改 |
| DELETE | `/api/child/{id}` | 删除（级联删除其筛查记录） |
| GET | `/api/child/{id}` | 详情 |

请求体（add/update）：
```json
{ "name": "小星星", "gender": "male", "birthDate": "2024-06-01", "avatar": "👶" }
```
> 年龄校验：出生日期距今 12-60 个月，否则返回 `本筛查适用于1-5岁（12-60个月）的儿童`。

## 三、问卷接口 `/api/questionnaire`（公开）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/questionnaire/list` | 问卷列表 |
| GET | `/api/questionnaire/{qid}` | 问卷详情（含题目+选项） |
| GET | `/api/questionnaire/default` | 默认问卷（id=1） |

题目选项结构：
```json
{ "id": 1, "qid": 1, "video_url": "/api/video/q1.mp4", "content": "...", "options": [{"value":0,"label":"经常会","score":0}], "sort": 1 }
```

---

## 四、视频接口 `/api/video`（公开，支持防盗链）

`GET /api/video/{文件名}` —— 支持 HTTP Range 分段播放（206 响应）。
防盗链：配置 `autism.allowed-referer` 后校验 `Referer`，留空则不校验。
> 视频文件存放于 `data/videos/` 目录，命名 `q1.mp4 ~ q20.mp4`。

## 五、答卷与报告接口 `/api/answer`（需登录）

### 5.1 提交答卷 `POST /api/answer/submit`
请求体：
```json
{
  "childId": 1,
  "qid": 1,
  "answers": [ {"questionId":1,"value":0}, {"questionId":2,"value":1}, ... 20题 ]
}
```
> 服务端按题库重新计分（防篡改），返回完整报告。

### 5.2 报告详情 `GET /api/answer/report/{id}`
### 5.3 我的历史记录 `GET /api/answer/history`
### 5.4 历史详情 `GET /api/answer/history/{id}`

报告响应：
```json
{
  "id": 1, "totalScore": 3, "maxScore": 60,
  "riskLevel": "low", "riskText": "低风险",
  "child": {...}, "createTime": "...",
  "answers": [{"questionId":1,"value":0,"label":"经常会","score":0}],
  "recommendations": ["建议1", "建议2", ...]
}
```

## 七、管理员接口 `/api/admin`（需管理员 token）

### 7.1 登录 `POST /api/admin/login`（公开）
`{ "username": "admin", "password": "admin123" }` → `LoginResponse`

### 7.2 数据总览 `GET /api/admin/stats`
```json
{
  "totalUsers": 1, "totalChildren": 1, "totalScreenings": 2,
  "riskDistribution": {"high":1,"medium":0,"low":1},
  "ageDistribution": [{"label":"2-3岁","count":1}],
  "completionRate": 100.0
}
```

### 7.3 列表查询
- `GET /api/admin/users` 用户列表（含儿童数、筛查数）
- `GET /api/admin/children` 儿童列表（含所属用户、筛查次数）
- `GET /api/admin/records` 筛查记录列表
- `GET /api/admin/records/{id}` 记录详情

### 7.4 CSV 导出 `GET /api/admin/export/{type}`
`type`：`users` / `children` / `records`
返回 `text/csv` 文件下载（含 UTF-8 BOM，Excel 中文兼容）。

## 八、错误码

| code | 含义 |
|------|------|
| 0 | 成功 |
| 1 | 业务错误 |
| 400 | 参数校验失败 |
| 401 | 未登录或登录过期 |
| 403 | 无权限 |
| 500 | 服务异常 |
